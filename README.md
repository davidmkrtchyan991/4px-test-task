## Установка для 4px

1. Клонируйте проект в локальную среду
2. Скопируйте файл '.env.example' с именем '.env' и задайте поля бд в этом файле
3. Запускайте команды по очереди
    - composer install
    - php artisan migrate
    - php artisan db:seed
    - npm install
    - npm run dev
4. Задайте локальный домен (корень на /public) и войдите в приложение данными из тз.
